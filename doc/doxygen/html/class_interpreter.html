interpret the text and store into info

This interprets FRD script, which is programming language to create fractal images and videos in Fractal Designer. After interpretation, is there is no syntax error, the information of the current script will be stores.

Parameters
textthe QString of Editor text
infothe current info
Returns
whether there is not syntax error
Return values
trueThere is no syntax error.
falseThere are syntax errors.
Remarks
This is a static function, you can call it by:
